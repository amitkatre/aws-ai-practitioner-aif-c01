## Quick start

1. Clone this repo (or download ZIP).
2. Open any `.html` file in **Chrome, Firefox, or Safari** (double-click or "Open with…").
3. **Study cheatsheet:** open `aif-c01-study-cheatsheet.html` and use **Print → Save as PDF** if you want a portable copy.
4. **Slides:** use **← →**, **Space**, **Home** / **End** to navigate decks.

## Publishing (optional)

These are **static files only**—no build step. You can:

- Use **[GitHub Pages](https://pages.github.com/)** (Settings → Pages → deploy from `main` / root) to serve the HTML.
- Point any static host at this folder.

If your employer restricts publishing to third-party SaaS, get **security / InfoSec approval** before enabling GitHub Pages or other external hosting.
